# 外部システム一覧

| ID | 区分 | 名称 | 概要・責務 | 本システムとの連携目的 |
|---|---|---|---|---|
| EXT-1 | 自社別システム | 商品マスタ管理システム | 商品仕様・特約・引受基準・保険料計算ロジックを管理する | 商品・特約選択および保険料試算で最新の商品仕様・計算ロジックを参照する(本システムは計算ロジックを保持しない) |
| EXT-2 | 自社別システム | 募集人管理システム | 募集人・代理店の登録情報と募集権限を管理する | 募集権限照合(対象顧客・商品を募集できるかの判定)のため募集人・代理店情報と権限を参照する |
| EXT-3 | 自社別システム | 契約管理システム | 契約成立後の保全(契約内容の維持・変更管理)を担う | 契約成立(計上)時に成立契約と確定顧客情報を引き渡し、保全業務へ連携する |
| EXT-4 | 自社別システム | 収納システム | 第2回目以降の継続的な保険料収納を担う | 契約成立後の継続収納へつなぐため成立契約情報を引き渡す(第一回保険料の収納は決済サービスが担う) |
| EXT-5 | 自社別システム | 帳票発送システム | 物理帳票の印刷・封入・郵送を担う | 保険証券 等の物理交付帳票の発送を委託し、発送連携受領確認をもって交付完了とする |
| EXT-6 | 外部サービス | 本人確認(eKYC)サービス | 本人特定事項の確認(取引時確認)を提供する | 申込時の取引時確認(本人特定事項の確認)を委託する |
| EXT-7 | 外部サービス | 反社チェックサービス | 反社会的勢力該当性の照合を提供する | 契約関係者(申込人・被保険者・受取人)の反社該当性照合を委託する |
| EXT-8 | 外部サービス | 電子署名サービス | 電子署名の付与・検証および真実性措置(タイムスタンプ)を提供する | 確定文書(申込書・告知書 等)への本人電子署名・検証・真実性措置付与を委託する(本システムは独自の署名/検証アルゴリズムを実装しない) |
| EXT-9 | 外部サービス | 決済サービス | 第一回保険料の収納手段(口座振替・カード・振込・コンビニ 等)を提供する | 第一回保険料の収納を委託し、手段別の成立条件で収納成立を検知する |
