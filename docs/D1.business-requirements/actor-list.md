# アクター一覧

## 本書について

* 本書は、Sample生命保険株式会社 個人保険新契約システムに関わるすべての **人間アクター** (関係者)を一覧化したドキュメントです。アクターは、システムを直接利用する者だけでなく、システムから出力された情報・帳票等を(他者経由を含めて)受け取る者や、システムを通じて影響を受ける者・統制する者も含みます。
* システム等の非人間アクターは本書では扱わず、後続の C4モデル(レベル1・2)で表現します。
* 本書はビジネス要件定義書(BRD)のスコープ・対象ユーザー概要を上流とし、後続の成果物であるプロダクト要求仕様書やユーザーストーリー一覧、C4モデルレベル1、C4モデルレベル2、L1シーケンス、L2シーケンスなど、様々な成果物から参照されます。
* 本書は、エンタープライズアーキテクチャフレームワーク TOGAF におけるActor Catalogに相当する成果物です。

## アクター分類の方針

アクターは以下3軸で分類します。

- **種別**: 人(個人または役割) / 組織(部門・社外団体)
- **境界**: 内(自社内) / 外(社外)
- **関与の仕方**: 直接利用 / 業務関係者(直接利用しないが業務上関わる) / 統制・監督
