# ドメイン共通要求仕様書

## 本書について

### 概要

<!-- HINT:
本書の位置づけ宣言。1〜2文でドメイン定義書との関係と本書のスコープ(3ドメイン以上の横断要求)を述べるに留める。
下流成果物では「本書について」等の前提説明を最小化する方針(想定読者・PRDとの関係解説等は省く)。
-->

本書は[ドメイン定義書](../domain-definition-document#一覧)で定義された各ドメインに対し、**3ドメイン以上で横断的に効く具体的な要求**を記載するドメイン要求仕様書です。

### 注記

<!-- HINT:
本書での How (具体実装手段)の扱い方針を1文で記述。
ビジネス・規制上譲れない具体水準(可用性目標値・監査ログ保持期間・準拠すべき業界基準 等)は本書に含めることを宣言する。
-->

本書では原則として How(具体的な実装手段)には踏み込みませんが、ビジネス上譲れない具体水準(例: 監査ログ保持期間・準拠すべき法的要件 等)はドメイン共通要求の不可分な構成要素として記載します。
